## 🚀 To clone this project using CLI:
1. Clone the project on your local environment.
```sh
git clone https://github.com/robin-dc/robindc-courses.git
```
2. Go to the project directory.
```sh
cd <project folder>
```
3. Open in Visual Studio.
```sh
code index.html
```
